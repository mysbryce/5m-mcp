Introduction

A FiveM resource that hands the keys to your AI agent — safely.

agent_api is a single FiveM resource that exposes a Model Context Protocol server. Once installed and started, any MCP-aware coding agent — Claude Code, Codex CLI, Cursor, Cline, your own — can develop, debug, and live-test FiveM resources against a running server.
